Какие проблемы решает svg-спрайт

  • меньше запросов к серверу (или вообще без них)
  • компактный код, легкое переиспользование иконок
  • возможность управлять цветом иконки
  • кеширование
  • семантика

Какие проблемы создаёт svg-спрайт

  • относительно сложная сборка
  • относительно сложно работать с цветом иконки

Плагины

  • Предобработка: SVGO
  • Webpack: svg-sprite-loader
  • Gulp: gulp-svg-sprite
  • Vue CLI: vue-cli-plugin-svg-sprite
  • Полифилл: svg4everybody
